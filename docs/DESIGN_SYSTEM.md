# TamTam Design System

## Color Palette

| Element | Hex Code | RGB | Usage |
|---------|----------|-----|-------|
| Primary Canvas | #FFFFFF | 255, 255, 255 | Main backgrounds, cards |
| Secondary Canvas | #F9F9F9 | 249, 249, 249 | Input fields, sections |
| Accent Red | #D32F2F | 211, 47, 47 | CTAs, highlights, actions |
| Deep Charcoal | #1A1A1A | 26, 26, 26 | Primary text |
| Muted Gray | #757575 | 117, 117, 117 | Secondary text, borders |
| Light Gray | #E0E0E0 | 224, 224, 224 | Dividers, borders |
| Success Green | #4CAF50 | 76, 175, 80 | Success states |
| Warning Orange | #FF9800 | 255, 152, 0 | Warning states |
| Error Red | #F44336 | 244, 67, 54 | Error states |

## Typography

### Font Family
- **Primary:** Inter (sans-serif)
- **Secondary:** Merriweather (serif) for headings

### Font Sizes
- **Display (Hero):** 48px, Bold
- **Heading 1:** 32px, Bold
- **Heading 2:** 24px, Semibold
- **Heading 3:** 20px, Semibold
- **Body Large:** 18px, Regular
- **Body Regular:** 16px, Regular
- **Body Small:** 14px, Regular
- **Caption:** 12px, Regular

## Components

### Button States

**Primary Button (CTA)**
- Default: #D32F2F, white text
- Hover: #B71C1C
- Active: #7B1D1D
- Disabled: #E0E0E0, gray text

**Secondary Button**
- Default: white background, gray border, gray text
- Hover: #F5F5F5 background
- Active: #E0E0E0 background

### Input Fields
- Background: #F9F9F9
- Border: #E0E0E0
- Focus: Border becomes #D32F2F
- Error: Border becomes #F44336

### Cards
- Background: #FFFFFF
- Border: 1px solid #E0E0E0
- Shadow: 0 2px 8px rgba(0, 0, 0, 0.1)
- Border-radius: 8px

## Spacing Scale

```
0.5rem = 8px
1rem = 16px
1.5rem = 24px
2rem = 32px
2.5rem = 40px
3rem = 48px
```

## Responsive Breakpoints

- **Mobile:** 0px - 640px
- **Tablet:** 641px - 1024px
- **Desktop:** 1025px+

## Icon Library

Use Feather Icons (https://feathericons.com/) for consistency.

- Size: 24px for navigation, 20px for inline
- Color: Match text color or use accent red for CTAs
